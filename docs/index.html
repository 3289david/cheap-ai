How it works

From install to coding in 3 steps

01

Deploy the server

Run docker compose up on any VPS. Server starts on port 3000 with the browser IDE bundled in.

02

Add your API key

Open /admin, add an OpenRouter or Anthropic key once. Your whole team shares it.

03

Pair CLI to web

Run cheap web. A 6-digit code appears. Type it in the browser. Paired.

04

Start building

Use cheap in any directory. The AI reads, writes, and runs code. Memory persists.

Install the CLI
npm install -g cheap-ai
Features

Everything you need. Nothing you don't.

Built for developers who want real control over their AI toolchain.

CLI-First Agent

Run cheap in any directory. The AI reads files, writes code, runs shell commands, and remembers your project.

Browser IDE

Monaco editor, file tree, real PTY terminal over WebSocket, AI chat — all in the browser. No extensions needed.

Self-Hosted

Your VPS, your data. One docker compose up and you're running. No telemetry. No SaaS fees.

Persistent Memory

The AI remembers your architecture, preferences, and past decisions in SQLite. Memory survives across every session.

Any AI Model

OpenRouter, OpenAI, Anthropic, Gemini, Ollama, or any OpenAI-compatible endpoint. Switch providers without changing code.

Admin Panel

Configure providers, manage users, set per-user rate limits, generate invite links, view usage stats — all in /admin.

Team Mode

One server, multiple users. 6-digit pairing code links CLI to web. Invite links with role assignment.

Docker Deployments

Deploy any project to a Docker container from the browser IDE. Live URL assigned automatically.

Real AI Tools

read_file, write_file, run_command, search_files, list_directory — the AI executes real actions, not simulations.

Multi-Provider

Your keys. Any model.

Configure any AI provider once in the admin panel. Your entire team shares one key — no per-user key management, no leaking credentials.

  • OpenAI — GPT-4o, o1, o3
  • Anthropic — Claude Sonnet, Haiku, Opus
  • Google Gemini — 2.0 Flash, Pro
  • OpenRouter — 300+ models via one key
  • Ollama — run local models on your server
  • Any OpenAI-compatible API endpoint
